Макет STREET88

Street88 - мой первый проект, который я создал, используя HTML, CSS и JavaScript. Этот сайт представляет собой одностраничный веб-сайт, демонстрирующий базовые навыки верстки и адаптивного дизайна. Он включает в себя элементы оформления, анимации и интерактивные функции, разработанные для улучшения пользовательского опыта.

Online store

В этом проекте я разработал полноценный интернет-магазин с использованием React. Сайт включает в себя каталог товаров, корзину покупок, форму заказа и интеграцию с API для обработки данных. Youtube

Этот проект представляет собой веб-приложение, аналогичное YouTube, разработанное на Laravel 8. Оно включает в себя функционал загрузки и воспроизведения видео, систему комментариев, подписку на каналы и систему рекомендаций. Проект демонстрирует мои навыки работы с серверной частью и интеграцией мультимедийного контента.

Ehson

Этот проект - онлайн-платформа для размещения объявлений, подобная OLX, созданная на Laravel 8. Пользователи могут публиковать и искать объявления по различным категориям, общаться с продавцами и покупателями через звонок. Проект включает в себя функции регистрации, авторизации, управления объявлениями.
